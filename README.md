# Tweeter Project

Tweeter is a simple, single-page Twitter clone made during the Lighthouse Labs coding bootcamp. 

You can make tweets and see them in a local feed. 

Dependencies: 
"body-parser": "^1.15.2",
"chance": "^1.0.2",
"express": "^4.18.2",
"md5": "^2.1.0"
